# Kyle McCool — Portfolio

Live: https://kyle-mccool.github.io/

Static portfolio / resume site for Kyle McCool. Co-Founder of Makko AI, Phantomline, and ProLocalBuilder. Based in Appleton, Wisconsin.

## Local preview

Just open `index.html` in a browser. No build step.

## Deploy

This repo is configured for GitHub Pages from the `main` branch root. Pushes to `main` auto-deploy in 1 to 2 minutes.

## Stack

Plain HTML and CSS. No frameworks, no JS, no build. Loads in under 50KB.
